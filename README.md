# Central Monitoring Panel

React frontend foundation for the Central Monitoring API and industrial IoT monitoring platform.

## Stack

- React, TypeScript, Vite
- Tailwind CSS and shadcn/ui conventions
- Radix UI, Lucide React
- TanStack Query and TanStack Table
- React Hook Form and Zod
- Axios, React Router, Recharts

## Commands

```bash
npm.cmd install
npm.cmd run dev
npm.cmd run build
npm.cmd run lint
```

## Environment

Copy `.env.example` to `.env.local` when backend integration starts.

```bash
VITE_API_BASE_URL=
```
